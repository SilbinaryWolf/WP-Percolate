Installing the Percolate Plugin for WordPress is easy, even if you haven 't done it before. WordPress plugins are fantastic because they allow you to add additional functionality and features which are not standard to a basic WordPress installation. If you are new to WordPress or Plugins, WordPress has some great background info on how to work with them on the site WordPress on Plugins.

The steps to adding the Percolate plugin are:

  1. Download the plugin
  2. Add New plugin
  3. Upload and Activate
  4. Complete Percolate Settings

1. Download the Plugin

The latest plugin can always be downloaded here. Save it to your computer, but don 't unzip it: https://github.com/percolate/WP-Percolate/archive/master.zip

2. Add New Plugin

Open your WordPress admin, navigate 'plugins ' on the left, and select 'Add New '.

3. Upload and Activate Plugin

Click 'upload ' at the top of the screen and locate the plugin that you downloaded above. It should only take a minute for the plugin to upload and you 'll see the confirmation message "Plugin installed successfully". Click on 'Plugins' on the left again to navigate to all of your WP plugins. Activate the Plugin: Click 'Activate ' under WP Plugin. Percolate Plugin Settings: The plugin has several settings you have to fill out to ensure the plugin is working properly. Click on 'Settings ' under WP Percolate.

4. Complete Percolate Settings

  • Percolate User Type - There are two user types the plugin is designed for; Single Percolate Users and Multiple Percolate Users. If there is more than one Percolate user, use the Multiple Users option. Percolate will provide this info.
  • Percolate API Key - The API Key authenticates your account and allows Wordpress to communicate with Percolate via the API. Percolate will provide this info. Website - This specifies which Percolate dot com posts will be imported. There will typically be only one option, select it.
  • Publish Status - This setting dictates whether Percolate posts will be imported as draft or live posts. We always recommend that posts be imported as drafts so they can be previewed and finalized before publishing to your Wordpress theme.
  • Default License Author - When a Wordpress author does not exist for a Percolate license author, posts from Percolate will be assigned from this Wordpress user. (i.e. if a new Percolate user creates a post but has not been mapped to a WP user)
  • License Authors - Allows you to map the Percolate license author with the Wordpress Author. There can be a unique Wordpress author for every Percolate license author publishing to WP. Click ''refresh'' to update WP with any new Percolate users that need to be mapped to a WP author. Otherwise the default in the step above will be used.
  • Category - This setting will assign a WordPress Category to all of the posts from Percolate (ex: "blog"). This can dictate where posts from Percolate will appear. Post Type - This setting allows you to specify what WordPress post type content from Percolate will be imported as. In most cases the default type ''post'' should be used.
  • Save - click 'Save Changes ' to ensure settings are updated. Support If you have any questions or troubles, please feel free to reach out to your Percolate client solutions manager anytime. They are more than happy to help you through the install process.
